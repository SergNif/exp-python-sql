Адаптивный многомодульный шаблон "Онлайн торговый центр" для Opencart 3.0.2.0

Внимание!!! Шаблон требует php версии 5.6 или 7.0 и ioncube на хостинге!!!

1. Установка:

1.1 Распаковать архив и скопировать из папки themes архива в корень сайта, где лежат аналогичные папки, следующие папки:
 - admin	
 - catalog
 - image
 - system


1.2 Для русификации вашего движка (если русификации еще нет) откройте папку languages_packet_for_opencart_3.0 в архиве шаблона и скопируйте папки admin и catalog поверх вашего сайта, нужные файлы сами скопируются по каталогам.




2. Если вы устанавливаете шаблон на чистый (без ваших товаров) движок:

2.1. Восстановить базу данных с файла sql/baza_for_opencart_3020.sql в админке вашего движка (СИСТЕМА - ИНСТРУМЕНТЫ - Бэкап/Восстановление - выбрать файл в папке  sql/baza_for_opencart_v3020.sql и нажать кнопку Восстановить).

После восстановления базы пароль и логин входа в админку изменится на admin и admin соответственно.

Важно знать, что база не импортируется, если префикс к БД, указанный Вами при установке сборки будет отличаться от установленного по-умолчанию "oc_". Если все-таки префикс отличается, то вам необходимо открыть файл sql/baza_for_opencart_3020.sql и вместо "oc_" указать свой префикс к таблицам. Там их много, поэтому лучше использовать программу notepad++ для автоматической замены префикса к таблицам.

После восстановления базы выполнять настройки вручную согласно п. 5 не обязательно - все само уже настроиться!

2.2. зайдите в админке по адресу Модули/Расширения - > Модификаторы -> в списке должно появится модицификация "Online ts modification for template Opencart 3.0" -> нажать кнопку Обновить в верхнем правом углу -> все, модификация должна работать!






3. Если ваш товар  уже есть на сайте и Вам надо установить шаблон на работающий сайт.

3.1. Зайдите в админке по адресу Модули/Расширения - > Установка расширений -> нажмите Загрузить -> выберите файл ocmod/alogin_admin_templates.ocmod.zip и ocmod/alogin_catalog_templates.ocmod.zip из папки ocmod архива шаблона -> далее в Модули/Расширения -> Модификаторы -> кнопка Обновить.

3.2. Выбрать шаблон в Модули/Расширения -> Модули/Расширения -> выбрать в списке Темы - нажать кнопку Редактировать -> выбрать папку шаблона 'online-ts' -> Сохранить

После этого зайдите в Система - Пользователи - Группы пользователей и выберите все для редактирования и просмотра для Главного администратора.







4. В данном шаблоне присутствует "Модуль настройки шаблона". Для его функционирования Вам необходимо зайти в раздел Модули/Расширения -> Модули/Расширения -> Модули и нажать кнопку Установить напротив Модуля настроек шаблона, нажать кнопку Изменить напротив него и затем ввести туда код, который Вы у меня запросите по контактам, написанным внутри этого модуля, после чего нажмете кнопку Применить. При установке на локальном хостинге модуль настроек шаблона работает без всяких ограничений и код не запрашивает.

Если после ввода секретного кода модуль не открылся, значит код введен неверный. При правильном коде откроется обширная панель настройки шаблона.

Если вам нужно, чтобы модуля располагались как в демо: то расположите модули, включив их сперва в настройках каждого добавляемого модуля, а затем добавьте их в схемах по адресу Дизайн -> Макеты в админке движка
